# PHP Docker 

> Very simple docker config for running php on localhost

## Requirements

* `Docker`
* `docker-compose`

### Tested on

* `MacOS 11.5.2`
* `Docker version 20.10.12`
* `docker-compose version 1.29.2`
* `bash-5.2`

## Install

1. Make sure your docker is running
2. Clone this repo
3. `$ docker-compose up`
4. Open your local URL: http://localhost:8000
